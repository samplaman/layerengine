Jitter Modulation

Compatibility

  • Windows: 10 / 11 (64-bit)
  • macOS: 11.0 or higher (Intel / Apple Silicon Native)
  • Linux: Ubuntu 22.04+, Debian, Arch, Fedora

Formats

  • VST3: VST 3.7.9 standard
  • AU: Audio Unit v2 (macOS only)
  • CLAP: CLever Audio Plug-in API
  • Standalone: Full JUCE Audio/MIDI Host application
